## 除法

对于除法 a / b = c ... R

求 R 的方式有`取模`、`取余`两种算法，存在些许区别

## 取模（计算机科学）

取模运算在计算 c 的值时，向负无穷方向舍入(floor()函数)

```python
7 % 4 # = 3（商 = 1 或 2，1＜2，取商=1）
-7 % 4 # = 1（商 = -1 或 -2，-2＜-1，取商=-2）
7 % -4 # = -1（商 = -1或-2，-2＜-1，取商=-2）
-7 % -4 # = -3（商 = 1或2，1＜2，取商=1）
```

## 取余（数学）

取余运算在取 c 的值时，向 0 方向舍入(fix()函数)

```js
r = 7 % 4; // = 3 (商 = 1)
r = -7 % 4; // = -3 (商 = -1)
r = 7 % -4; // = 3 (商 = -1)
r = -7 % -4; // = -3 (商 = 1)
```
